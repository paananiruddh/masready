
When enabled:

* no live API execution
* no Maximo connection testing
* no API key entry
* no editable Maximo URLs
* no live write/update/delete actions
* no user-provided endpoints
* no secret exposure
* seed/demo data only
* integrations visually visible but disabled

Show disabled controls as:

* visible
* informative
* non-interactive

Example labels:

