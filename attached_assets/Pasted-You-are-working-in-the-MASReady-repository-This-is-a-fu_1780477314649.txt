CORE PRODUCT DECISIONS TO IMPLEMENT
===================================

Implement the following product architecture decisions:

1. MASReady has three commercial layers:

   * Template Layer
   * Module Layer
   * Hydration Layer

2. MASReady has five technical layers:

   * Base Template
   * Module Registry
   * Canonical Data Model
   * Hydration / Adapter Layer
   * Evidence / Governance Layer

3. The existing MAS9POWER demo must be treated as:

   * a working functional demo company
   * a proof that the product works
   * a source from which reusable template areas can be extracted
   * a source of demo data patterns
   * not the reusable template itself

4. The clean reusable template must not contain:

   * MAS9POWER-specific company data
   * real customer data
   * hardcoded customer names
   * hardcoded demo assumptions
   * credentials
   * one-off demo-only logic

5. Modules are independently enableable but not necessarily independently hydrated.

6. Modules must not directly call each other.

7. Modules consume:

   * requires.data = canonical facts from hydration
   * consumes.signals = outputs from other modules

8. Modules produce:

   * outputs
   * signals
   * evidence
   * confidence metadata
   * completeness metadata

9. The canonical data model is the key contract between hydration and modules.

10. Governance must be enforced through configuration and validation, not just documentation.

11. DEMO and LIVE modes must be separated:

* DEMO mode can use MAS9POWER fictional/synthetic data.
* DEMO mode must not access live connectors.
* LIVE mode must reject MAS9POWER fictional/synthetic data.
* LIVE mode is read-only by default.
* SQL execution is disabled by default.
* Maximo/MAS mutation is disabled by default.
* Write-back is disabled by default.

12. Evidence packs must include:

* provenance
* source checksums
* confidence
* completeness
* what MASReady touched
* what MASReady did not touch
* generated hash/checksum
