# Google Search Console Setup

## Verification

Add the Google site verification meta tag to `index.html`:

```html
<meta name="google-site-verification" content="YOUR_TOKEN_HERE" />
```

Or upload `googleYOUR_TOKEN_HERE.html` to the repository root.

## Sitemap Submission

Submit `https://softwarecomparereview.github.io/masready/sitemap.xml` in Google Search Console.

The sitemap covers all public pages including design variants, simulator, and compare pages.

## Canonical URLs

All pages use canonical tags pointing to `https://softwarecomparereview.github.io/masready/` base.
